General Information:

What is the game

This game is a visual novel making use of Live2D character models to allow greater player immersion.

What is a Visual Novel (VN)

Visual novels are easily distinguishable from other games by their minimalistic gameplay, the main method that a player interacts throughout the story is mostly with a single mouse-click. This is done when making narrative choices throughout the course of the game.

These types of games also feature another notable characteristic, the emphasis on the prose, seeing as the narration is delivered through text. This makes playing a visual novel similar to reading a book.

Most visual novels have multiple storylines which lead to more than one ending to the story. This is done by incorporating the mechanic of using irregularly-occurring multiple-choice decision points, where the player selects what course to take the game in.

Most visual novels make use of still (static) images for character models and background scenes. This allows high quality images to be used when running the game, even on low-end gaming systems.

What is Anime?:

Anime is hand-drawn and computer animation originating from Japan. Anime, a term derived from the English word animation, is used in Japanese to describe all animation, regardless of style or origin. Outside of Japan, anime can refer either to animated works produced in Japan, or to their common visual style, which has since been adopted by a minority of works produced in other countries.

What is Live2D?:

Live2D is a technique of generating animated 2D graphics, usually anime-style characters, using layered, continuous parts based on a single illustration, without the need of animating frame-by-frame or creating a 3D model.

Getting Started:

Installing the game

The game is installed automatically after being downloaded from the Steam game launcher.

Uninstalling the game

There are two methods to uninstalling this game:

  1. The game can be uninstalled using the Steam app by finding the game in your local Library, right-clicking and hovering over the Manage option. The Uninstall option should be visible. This can be used to uninstall the game.

  1. The game can be uninstalled by deleting the game folder on your local storage drive. Since all the data, savegames and settings files are stored within this folder no other files regarding this game will be found after deleting this folder.

Updating the game

The game will irregularly receive updates as deemed necessary after launch. These updates will be available to download from the Steam game launcher Library. After downloading the update it will be automatically installed.

Starting the game

The game is launched either through the Steam Library by selecting the game title and pressing the Play option, or via the game local folder by double-clicking the Game.exe file.

Using the System:

Playing the game for the first time

  • Dialogue input events

These events are the main interaction that players will be making use of throughout the course of the game. This allows the player to choose a specific dialogue response based on a given list of options. Depending on the current story chapter, scene and characters spoken to it could potentially lead to a different story ending/resolution than a different option would have led to.

  • Character Dialogue differences

The dialogue text displayed for the various characters can differ in colour and font-style. This is used to further familiarise the player with the different characters and have them be more aware of the character currently speaking by having them be displayed uniquely at times.

  • Main Menu Title Screen

  • New Game

Starts a new game from the beginning of the game story. Additionally, the first time the Save button is pressed in-game during this game playthrough will have the next empty save file slot be used for the new save file.

  • Load Game

Displays a screen with save file slots to choose a previously created save file from.

  • Settings

Leads to a screen showing different setting categories to select from, namely:

  • General
  • Display Mode

Displays the game in either Fullscreen or Windowed mode.

  • Screen Adjustment

Allows the game to be enlarged based on screen resolution ratios or stretched to fit the current display width and length.

  • Confirmation

Enables/Disables additional in-game confirmation for dialogue choices selected.

  • Message

  • Message Speed

Adjusts the speed at which the dialogue text is displayed on screen.

  • Auto-Message

Enables/Disables automatic dialogue progression when not presented with a dialogue input event, for example: a 3 paragraph character rant is originally presented as 3 separate paragraphs with each completed paragraph requiring the player to click for the next paragraph to replace the one currently displayed. By enabling this option the need for clicking is removed in such instances by having the paragraphs automatically replace each other.

  • Auto-Message Duration

Adjusts the duration the completed text is displayed on screen before replacing it when no dialogue input event is presented.

  • Audio

  • Music Volume

Adjusts the volume of the music used throughout the game.

  • Sound Effects Volume

Adjusts the volume of the sound effects used throughout the game.

  • Quit

Closes the game to return users to their desktop screens.

In-game menu buttons

  • Skip

Button used to skip the current chapter scene if it has been seen already. This is mostly used when a user exits the game without saving their current progress first.

  • Log

This button is used to display a chat log of all the recently displayed character dialogue text. This is useful for refreshing your memory of any previously made game progress which might have been forgotten over time.

  • Save

Used to save a user’s current game progress to be used as a starting point for continuing the game from if the user chooses to:

  1. Quit the current game in progress to close the program.

  1. Wants to have a save point to test different end-game routes at a later date.

  • Load

This button is used to load a previously created game save file (created via the Save button as mentioned above). This can be used immediately during game playthrough to load save points.

  • Title

This is used to immediately return to the game title screen/main menu. This does not automatically create a savepoint for the current game progress, which means if a user wants to keep their game progress and return to the game title screen users first need to press the Save button and immediately after saving press this button to ensure no additionally-made progress is lost.

Game Character List

  • Takashiro, Touya 

A 2nd-year student at Kamiyama High School and the game protagonist. He is friendly, helpful, always puts others first and doesn’t care about receiving praise for what he does. While he is a member of the Cooking Club, he also likes to assist in the school library at times. His favourite sport is basketball.

  • Makabe, Raku

A 2nd-year student at Kamiyama High School and Touya’s best friend. He loves to joke around but is quite serious regarding sport. He is always available when Touya needs help or even just wants to talk with someone. Raku is also a member of the school basketball team.

  • Takashiro, Aki

Currently a middle school student and Touya’s younger sister. She cares quite a lot about her brother, to the point of others saying she cares too much. In the morning she takes responsibility for making breakfast just to let Touya sleep a bit more, seeing as he isn’t much of a morning-person. She is overprotective of Touya, but this goes both ways since he is also overprotective when it comes to her. They often converse and are quite relaxed around one another.

  • Satou, Rinka

A 1st-year student at Kamiyama High School and a childhood friend of Touya. She mainly uses the nickname Ri, and is close friends with Yui. Rinka is a member of the Cooking Club and practices kendo externally from school. She is very fond of Touya because during a previous national kendo tournament she received an injury that risked having her need to quit kendo completely. While she was recovering, Touya took the initiative to visit her daily and keep her spirits up.

  • Hanasaka, Moka

A 3rd-year student at Kamiyama High School and the current president of the Cooking Club. She feels indebted to Touya because he offered to join the club the previous year while it was at risk of being shut down due to lack of members. After Touya had joined, two additional members also decided to join the club. Moka is a friendly and energetic upperclassman who loves to be outdoors and try new experiences.

  • Tendou, Yui

A 1st-year student at Kamiyama High School. Yui is shy and prefers spending most of her time in the school library. Touya managed to befriend her and has since then assisted her in slightly overcoming her shyness, enabling her to manage making a few more friends. She decided to join the Cooking Club and additionally assists in the school library at times.

  • Maki, Noriko

A 3rd-year student at Kamiyama High School and the current Student Council President. She is quite fond of Touya, this stems from the fact that he is always willing to help the Student Council with any problems they are facing, such as brainstorming ideas for new events. She likes Touya because of the ever-present smile on his face, seeing him always improves her mood since he never seems to be depressed.

  • Himegami, Azuki

A 2nd-year student at Kamiyama High School with a Tsundere personality-type. She can be seen as uptight by others, in reality she is intelligent and responsible. Earning her the position of class representative. She is often being teased, leading to Touya checking in on her regularly. This interaction usually ends with Azuki getting flustered and blushing dramatically. She likes to tell others “It’s not like I like him or anything...”.

  • Okazaki, Hotaru

A 3rd-year student at Kamiyama High School and the Vice-President of the Student Council. She is extremely overprotective of her best friend, Noriko, and is quite clumsy. Leading to her ending up in strange situations with Touya and accusing him of purposefully being the cause.

  • Kojima, Hikari 

Currently employed as a teacher at Kamiyama High School. She is very relaxed and laid-back but is also considered a clutz. She is caring towards her students and interacts with them quite frequently, leading to most of her students calling her by the nickname, Hiki. Hikari has a positive relationship with members of Touya’s family and thus knew him from a young age. She began making a habit of visiting the school’s Cooking Club at the end of school operating hours to taste-test some of their newest creations and to converse with Touya.
